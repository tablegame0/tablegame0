# 🎮 Interactive Bingo Card Generator

A modern, responsive Bingo Card web application built with **Tailwind CSS** and custom **CSS3** animations.

### ✨ Key Features
* **Dynamic Shuffling:** Automatically randomizes card items on every page load or reset.
* **RGB Border Animation:** Features a smooth, multi-shade moving RGB border effect.
* **Custom Background:** Supports custom image backgrounds within the card layout.
* **Fully Responsive:** Optimized font sizes and layouts for both Mobile and Desktop devices.
* **Performance:** Lightweight and fast, built using Vanilla JavaScript and Tailwind CSS.

### 🛠️ Tech Stack
* **HTML5** & **Tailwind CSS** (Utility-first styling)
* **CSS3** (Custom Animations, Flexbox, Glassmorphism)
* **JavaScript** (Vanilla JS for core logic)

### 🚀 How to Use
1. Enter your list of items or names in the input field.
2. Click **Submit** to generate your custom-styled Bingo card.
3. Use it on any device; the layout adjusts automatically.
